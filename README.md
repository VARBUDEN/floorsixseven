## 📖 О ИГРЕ

Вы — администратор смотровой площадки на 67-м этаже. Ваша задача — работать 12 часов в день, балансируя между выполнением обязанностей и желанием отдохнуть. Смотрите в телефон, чтобы восстановить силы, но не попадитесь проверяющим. Копите зарплату, улучшайте репутацию и держитесь как можно дольше, пока вас не уволят.

## 🎮 ГЕЙМПЛЕЙ

### Основные механики

| Механика | Описание |
|----------|----------|
| 📱 **Телефон** | Смотрите вниз, чтобы восстановить стамину. Тратит заряд |
| ⚡ **Стамена** | Тратится на рабочих точках. При 0 — день заканчивается |
| 😠 **Гнев** | Растёт в зонах проверки. При 100% — досрочный конец дня |
| 💀 **Увольнение** | При 3-4 выговорах (в зависимости от персонажа) игра заканчивается |
| 💰 **Зарплата** | Начисляется каждый час. Бонус от рабочих точек |
| 🗓️ **Расписание** | Нужно стоять на определённых точках в определённое время |
| 💬 **Диалоги** | Утренние просьбы от коллег. Влияют на репутацию |
| 📊 **Репутация** | Влияет на качество расписания (выше репутация → лучше точки) |
